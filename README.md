# 🛒 VNTG Hub - E-commerce de Coleccionables

## 📄 Descripción del Proyecto
Plataforma de comercio electrónico de alto rendimiento diseñada para coleccionistas de cultura pop. Este sistema utiliza una arquitectura moderna de **Next.js** para garantizar velocidad, estabilidad y una experiencia fluida tanto en PC como en móviles.

## 🚀 Stack Tecnológico
* **Frontend:** [Next.js 15+](https://nextjs.org/) (App Router) + TypeScript.
* **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/) (Mobile-first).
* **Iconos:** Lucide React.
* **Temas:** next-themes (Soporte Modo Claro/Oscuro).
* **Despliegue:** [Vercel](https://vercel.com/).

## 📁 Estructura y Setup
El proyecto se divide en `/client/vntg-hub` y `/server`.

**Para empezar:**
1. `cd client/vntg-hub`
2. `npm install`
3. `npm run dev -- --turbo`
