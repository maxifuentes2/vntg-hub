# 🛒 VNTG Hub - E-commerce de Coleccionables

## 📄 Descripción del Proyecto
Plataforma de comercio electrónico diseñada para coleccionistas de cultura pop. Este sistema utiliza una arquitectura moderna basada en React.js para garantizar una experiencia fluida y reactiva tanto en PC como en dispositivos móviles.

## 🚀 Stack Tecnológico
* Frontend: React.js + TypeScript.
* Estilos: Tailwind CSS v4 (Mobile-first).
* Iconos: Lucide React.
* Temas: Soporte para Modo Claro/Oscuro.
* Herramienta de Construcción: Vite.

## 📁 Estructura y Setup
El proyecto está organizado en dos carpetas principales: `/client/vntg-hub` y `/server`.

Instrucciones para iniciar el proyecto:

1. Servidor (Backend & DB):
   cd server
   npm install
   npm run dev

2. Cliente (Frontend):
   cd client/vntg-hub
   npm install
   npm run dev

Nota: Asegúrate de ejecutar ambos comandos en terminales separadas para mantener activos tanto el backend como la interfaz de usuario.

## ⚠️ Reglas
1. Ramas: No subir cambios directamente a main. Usar el formato de rama dev/tu-nombre.
2. Pull Requests: Todo cambio debe ser revisado y aprobado antes de integrarse.
3. Seguridad: No subir bajo ninguna circunstancia el archivo .env al repositorio.
